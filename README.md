# BI Analytics Platform V8

نسخة مطورة من منصة ذكاء الأعمال وتحليل البيانات المؤسسية.

## أبرز التحديثات في V8

1. تحسين صفحة **مجموعات جلسات التحليل**:
   - بطاقات تنفيذية لكل نوع تحليل.
   - إظهار عدد الجلسات، السجلات، والمؤشر الرئيسي.
   - إضافة شرح/بيان مختصر بجانب الأرقام لتوضيح معنى كل مؤشر.

2. تطوير **التقرير التنفيذي**:
   - استرجاع التقرير التفصيلي الاحترافي.
   - إضافة فلتر Check Box لاختيار أكثر من جلسة/ملف ضمن التقرير.
   - إضافة ملخص تنفيذي، تعريف المؤشرات، جداول الأداء، الخدمات الأعلى والأقل رضا، التحليل النوعي، وخطة العمل.
   - دعم تصدير التقرير إلى Word.

3. تحسين **التحليل النوعي**:
   - إزالة التكرار من الملاحظات التفصيلية.
   - استبعاد العبارات غير المفيدة مثل: لا يوجد، لا توجد، none، n/a.
   - تغيير مسمى “نقاط التميز” إلى “ملاحظات الثناء والتجارب الإيجابية”.
   - توحيد حجم قسمي الملاحظات الإيجابية والتحسين/الشكاوى.

## رفع المشروع على GitHub Pages

1. فك ضغط الملف.
2. ارفع محتويات المجلد مباشرة إلى Repository بحيث يكون `index.html` في الجذر.
3. من GitHub افتح: Settings → Pages.
4. اختر: Deploy from a branch.
5. اختر: Branch = main، Folder = /root.
6. اضغط Save.

## تحديث V13 - حل مشكلة QuotaExceededError
تم تعديل آلية الحفظ بحيث لا يتم نسخ البيانات الأصلية الكبيرة داخل LocalStorage. تستخدم النسخة الآن IndexedDB للتخزين الموسع، مع حفظ ملخص صغير فقط في LocalStorage. هذا يعالج رسالة:
`Failed to execute 'setItem' on 'Storage': exceeded the quota`.

ملاحظة: إذا ظهرت بيانات قديمة بعد الرفع، افتح أدوات المتصفح وامسح LocalStorage القديم للموقع مرة واحدة أو اضغط إعادة ضبط من البرنامج.
