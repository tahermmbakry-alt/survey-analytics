# منصة تحليل استطلاعات الرأي ورضا المستفيدين

مشروع جاهز للنشر على GitHub Pages بعد فصل ملف HTML الكبير إلى ملفات منظمة.

## التشغيل المحلي
افتح `index.html` مباشرة في المتصفح.

## النشر على GitHub Pages
1. أنشئ Repository جديد في GitHub.
2. ارفع محتويات هذا المجلد كما هي، بحيث يكون `index.html` في الجذر مباشرة.
3. افتح Settings > Pages.
4. اختر Deploy from a branch.
5. اختر Branch: main و Folder: /root.
6. اضغط Save.

## هيكل المشروع
- `index.html`: الواجهة الرئيسية.
- `css/style.css`: تنسيقات الواجهة.
- `js/app.js`: منطق التطبيق والتحليل والتصدير.
- `data/initial-data.js`: بيانات البداية المفصولة عن HTML لتقليل حجم الصفحة الرئيسية.
- `.nojekyll`: لتشغيل الملفات على GitHub Pages دون معالجة Jekyll.

## ملاحظة مهمة
يفضل مستقبلاً عدم إضافة البيانات داخل HTML. أضف ملفات Excel من داخل التطبيق أو خزّن التحليلات كملفات JSON منفصلة داخل مجلد `data`.
