# منصة تحليل استطلاعات الرأي ورضا المستفيدين

مشروع جاهز للنشر على GitHub Pages.

## أهم الخواص
- استيراد ملفات Excel لمنصتي إجادة وتجاوب.
- منع استيراد الملف نفسه أو المحتوى نفسه مرة أخرى.
- رسالة نجاح تعرض ملخص الملف المستورد: المصدر، عدد السجلات، عدد الخدمات، CSAT و NPS.
- تصدير بيانات البرنامج بصيغ XLSX و CSV و JSON.
- حذف أي ملف تم استيراده سابقاً مع حذف محتوياته وإعادة احتساب جميع المؤشرات.
- تقرير تنفيذي قابل للتصدير إلى Word مع رسوم بيانية مختصرة.

## طريقة الرفع على GitHub Pages
1. فك ضغط ملف المشروع.
2. افتح GitHub وأنشئ Repository جديد، ويفضل أن يكون Public.
3. اضغط Add file ثم Upload files.
4. ارفع محتويات المجلد مباشرة، وليس المجلد نفسه. يجب أن يظهر index.html في جذر المستودع.
5. من Settings ثم Pages اختر:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
6. اضغط Save.
7. سيظهر رابط الموقع بعد دقائق بصيغة:
   https://USERNAME.github.io/REPOSITORY/

## ملاحظة
المجلدات الفارغة قد لا تظهر في GitHub، لذلك تم وضع ملف .keep داخل assets و sessions و templates.
